# ReaShader

_a VST Video Processor for Reaper_

<br>

## Dependencies

<br>

Directory structure

```
+ reashader
+ vst-sdk-bundle
```

Clone the vst-sdk-bundle repo next to reashader repo.
<br>
The repo contains the fixed libraries/sdk to make the vst work.
